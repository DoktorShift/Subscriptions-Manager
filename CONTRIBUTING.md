# Contributing

Thank you for your interest in Subscriptions Manager!

## Reporting Issues

Found a bug or have a feature request? Please open an issue!

### Before Opening an Issue

1. Check existing issues to avoid duplicates
2. Include your LNbits version
3. Describe the problem clearly

### Bug Reports

Please include:

- **Description**: What happened?
- **Expected behavior**: What should have happened?
- **Steps to reproduce**: How can we recreate the issue?
- **LNbits version**: (e.g., 1.4.0)
- **Browser/OS**: (if relevant)
- **Screenshots**: If applicable

### Feature Requests

We welcome ideas! Please describe:

- **The problem**: What are you trying to solve?
- **Proposed solution**: How do you envision it working?
- **Alternatives considered**: Any other approaches you've thought of?

## Questions & Support

- For setup help, see our [Guide](GUIDE.md)
- For complete documentation, visit [docs-subscriptions.netlify.app](https://docs-subscriptions.netlify.app)

## Code Contributions

The extension source code is in a private repository. If you're interested in contributing code, please reach out via GitHub issues to discuss.
